This repository contains SageMath codes for searching for sides to $c,e,g,i$.

[This folder](https://github.com/JAgbanwa/Magic-square-of-squares-paper/tree/main/Verification%20of%20magic%20squares%20) contains the verification of correctness 
of all the examples of magic squares in [this paper].

This is also a Lean formalization of these results: [MagicSquareOfSquares.lean](https://github.com/JAgbanwa/miscellaneousmathproblems/tree/main/magic-square-of-squares). The only ``sorry`` part of the formalization pertains to the open problem itself i.e. all 9 cells of a magic square of squares being perfect squares.
